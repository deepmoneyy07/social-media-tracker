This file provides guidance to Claude Code (claude.ai/code) when working with code in this repository.

## Overview

This is a GitHub Pages documentation site for Vantage Circle's Rewards and Recognition (R&R) platform. It uses the Jekyll `cayman` theme and is published at `https://vcadmin0001.github.io/`.

## Structure

- `README.md` — Main documentation covering the full R&R admin portal: Overview analytics, Reports, Configuration (badges, awards, budget), and Company management (employees, departments, roles)
- `overview.md` — Standalone overview section
- `_config.yml` — Jekyll config (theme: jekyll-theme-cayman)

## Editing content

All documentation is plain Markdown with inline HTML for lists and images. Screenshots are hosted on GitHub's CDN (`user-images.githubusercontent.com`). When adding new sections, follow the existing heading hierarchy: `##` for top-level sections, `###` for subsections.

The live site is served directly from the `main` branch — changes pushed there deploy immediately via GitHub Pages.
